Linear Gradient Background

A linear gradient creates a smooth transition between colors along a straight line. This section demonstrates a linear gradient that flows from purple to pink, creating a beautiful and modern aesthetic.

Key Features:

  • Direction: Diagonal (135 degrees)
  • Colors: Purple (#667eea) to Pink (#f093fb)
  • Effect: Fixed attachment for parallax scrolling
  • Smooth color transitions throughout

Notice how the background stays fixed as you scroll, creating a parallax-like effect thanks to the background-attachment: fixed property!

Radial Gradient Background

A radial gradient creates color transitions that radiate outward from a center point, forming circular or elliptical patterns. This section showcases a vibrant radial gradient emanating from the center of the screen.

Key Features:

  • Shape: Circular gradient pattern
  • Colors: Orange (#ff6b6b) to Deep Purple (#4ecdc4) to Navy (#1a1a2e)
  • Origin: Center of the element
  • Creates depth and focal point

Radial gradients are perfect for creating spotlight effects, button backgrounds, or adding visual interest to hero sections!
